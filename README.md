# Middleware Technology 

## Second Attempt at the PVL

This project is a second attempt at the PVL. It involves the following technologies:

- **Frontend:**
  - Java
  - HTML
  - Thymeleaf (for the graphical representation)

- **Backend:**
  - Spring Boot

- **Persistence:**
  - H2 database
